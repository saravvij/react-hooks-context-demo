# State Management
Project explains state management using react context hook instead of using Redux.

### Examples:
1. Counter Example
    - Global state management through context
    - No reducer used
    - Good for simple use case

2. Contacts Example
    - Global state management through context
    - Uses reducer functions
    - Great for complex use cases
